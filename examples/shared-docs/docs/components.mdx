## Custom component flattener

`<Regulation>` is **not** part of Leadtype's naming contract — it's an app-defined
component. `examples/shared-docs/flatteners.ts` teaches the agent pipeline how to
flatten it with `defineComponentFlattener`, wired in via the config `flatteners`
field. In the browser it renders as a styled note; in the generated `.md` it
becomes a labeled blockquote.
